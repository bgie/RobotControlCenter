Python basis

  • Naar het uitvoerscherm schrijven: print(string)
  • Als-dan if conditie:
       code indien conditie waar
  • Als-dan-anders if conditie:
       code indien conditie waar
    else:
       code indien conditie niet waar
  • Variabele toewijzen variable = waarde
    • Een variable is een beetje ruimte in het computergeheugen waar je iets kan opslaan. Je geeft die ruimte een naam, de variabele naam, en je mag die naam zelf kiezen. Een duidelijke naam is wel aangeraden! Een naam moet uit 1 woord bestaan, maar met _ (underscore) kan je woorden aan mekaar plakken.
    • De waarde zijn de gegevens die je in de variabele wil steken. In een variabele kan je verschillende soorten gegevens opslaan. Dit noemen we het type van de gegevens. Zo zijn er bijvoorbeeld getallen of strings, en objecten.
    • Een string is een reeks letters, cijfers of symbolen. Wil je een string maken in je code, dan moet die tussen aanhalingstekens. In python kan je zowel enkele '' of dubbele "" aanhalingstekens gebruiken, maar je mag niet combineren in dezelfde string. Enkele voorbeelden: begroeting = 'hallo'
      robotnaam = "R2D2"
      het_beest = '666'.       Ook al bestaat een string enkel uit cijfers, het blijft een string als hij tussen aanhalingstekens staat. Wil je er mee rekenen, dat moet je de string omzetten naar een getal.
    • Een getal bestaat enkel uit cijfers en relevante symbolen (minus teken, decimaal teken...), en staat NIET tussen aanhalingstekens zoals een string. Met getallen kan je ook rekenen. Enkele voorbeelden: geld_op_mijn_rekening = 0
      minuten_per_dag = 60 * 24
    • Een object is een verzameling gegevens waar ook code aan hangt. Als je een object in een variabele steekt, kan je code die bij het object hoort uitvoeren. Dat doe door een . (punt) tussen de variabele naam en de naam van de code functie te zetten. Bijvoorbeeld: mijn_robot.haal_ne_cola()
  • Een vertraging (wachten) kan je inbouwen met het commando: time.sleep(seconden)
    • seconden is een getal. Het mag een komma-getal zijn. Merk op: in het Engels wordt een . (punt) gebruikt in plaats van een komma.
    • Dit commando zit in de time bibliotheek. Je moet die inladen. Dat doe je met volgende instructie aan het begin van je programma: import time

Robot verbinding

Inladen van de bibliotheek om met de robot te verbinden doe je met volgende instructie:

from robots import *

Deze import zet je best helemaal vanboven aan het begin van je programma.

Verbinden met de robot kan met deze code:

with Robot(naam) as variable

Parameters:

  • naam is de naam/nummer van de robot waarmee je wil verbinden. Dit mag een getal zijn of een string.
  • variable is een variabele die wordt aangemaakt. Via deze variabele kan je de robot instructies geven.

Voorbeeldcode om met robot 3 te verbinden en hem te laten draaien:

from robots import *

with Robot(1) as mijn_robot:
   mijn_robot.turn(70)

Rijden

  • Er zijn verschillende commandos om de motoren aan te sturen. Na zo'n commando gaat het programma onmiddellijk verder met de volgende instructie. Als je wil dat de robot een tijd blijft rijden voor je een ander commando stuurt, gebruikt dan een vertraging (time.sleep)
  • Het is mogelijk dat de robot bij lage snelheden niet beweegt, maar enkel piept. Gebruik dan een hogere snelheid.
  • Er zit een veiligheid in de software die zorgt dat de robot nooit langer dan meerdere seconden blijft rijden, tenzij je het commando herhaalt.
  • mijn_robot is telkens de naam van de variabele gebruikt bij het verbinden met de robot. Dit kan een andere naam zijn als je een andere variabele hebt gebruikt bij het verbinden.
  • Rechtdoor rijden kan met: mijn_robot.drive(snelheid)
    • snelheid is een percentage van -100 tot 100. Positieve getallen laten de robot naar voor rijden, negatieve naar achter.
  • Ter plekke draaien kan met: mijn_robot.turn(snelheid)
    • snelheid is een percentage van -100 tot 100. Positieve getallen laten de robot met de klok mee draaien, negatieve tegen de klok in.
  • Stoppen kan met: mijn_robot.stop()
  • De motoren apart aansturen kan met: mijn_robot.motors(snelheid_links, snelheid_rechts)
    • snelheid_links en snelheid_rechts zijn percentages van -100 tot 100. Positieve getallen laten een motor naar voor rijden, negatieve naar achter. Een flauwe bocht naar links kan je bijvoorbeeld maken door de linkse motor wat trager te zetten dan de rechtse: mijn_robot.motors(50, 100)
  • Draaien met een bepaalde hoek: mijn_robot.turn_by_angle(hoek)
    • hoek is een hoek van -180 tot 180 graden. Positieve hoeken zijn met de klok mee.
  • Draaien tot de robot naar een bepaalde richting kijkt: mijn_robot.turn_to_direction(richting)
    • richting is een hoek van 0 tot 360 graden waar de robot naar moet kijken.
      • 0 graden is links op het camerabeeld
      • 90 graden is boven op het camerabeeld
      • 180 graden is rechts op het camerabeeld
      • 270 graden is onder op het camerabeeld

Gamepads

Warning! Under development!

Als er gamepads zijn aangesloten, worden deze aan de robots gekoppeld volgens nummer.

Opvragen van de x of y as van de gamepad:

mijn_robot.gpadx
mijn_robot.gpady

Waardes zijn <0, 0 of >0 afhankelijk van de positie van de as.

De gamepads worden samen met de camera uitgelezen. Om in een lus telkens de huidige waarde op te vragen, kan je volgende code gebruiken:

while mijn_robot.camera.read():   print("x:", mijn_robot.gpadx)   print("x:", mijn_robot.gpady)
